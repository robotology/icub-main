# 1. Prerequisite

- Compile as static library `ICUB_SHARED_LIBRARY              OFF`
- Select UNITTEST flag  ` BUILD_TESTING                 ON`

# 2. Test execution

The test will be executed with the following:

```bash
cd build
make test
```
